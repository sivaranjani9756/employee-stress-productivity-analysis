# Employee Stress & Productivity Analysis

## Project Overview
This project analyzes employee stress levels and their impact on productivity using real-world HR data.  
The goal is to help organizations identify stress drivers, productivity risks, and actionable workforce insights.

## Business Objectives
- Identify key factors contributing to employee stress
- Analyze productivity patterns across roles and work conditions
- Support HR teams with data-driven decision-making
- Provide insights for improving work-life balance

## Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- SQL (MySQL)
- Power BI
- Jupyter Notebook

## Key Analysis Performed
- Data cleaning and preprocessing
- Handling missing values and outliers
- Exploratory Data Analysis (EDA)
- Stress score and productivity trend analysis
- Correlation analysis between stress, sleep, work hours, and WFH
- SQL-based querying for HR insights
- Interactive Power BI dashboard development

## Key Insights
- High work hours and overtime strongly correlate with increased stress
- Employees with balanced WFH options show higher productivity
- Sleep hours significantly impact productivity scores
- Certain job roles experience higher burnout risk

## Dashboard Highlights (Power BI)
- Stress & Productivity KPIs
- Job Role-wise Stress Analysis
- Burnout Risk Segmentation
- Work-Life Balance Indicators

## Outcome
This analysis enables HR and leadership teams to:
- Detect high-risk employee segments
- Improve workforce productivity
- Design better stress management strategies
